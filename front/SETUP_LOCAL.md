### Configuration et lancement du Frontend (React Native - Expo)

Pour exécuter l’application mobile localement (téléphone, émulateur Android/iOS ou web), suivez ces étapes :

#### Prérequis

- Node.js ≥ 18
- Expo CLI installé globalement ou usage de `npx expo`
- Application Expo Go sur votre téléphone (iOS ou Android)

#### Installation des dépendances

```bash
cd front
```

1. Demarrez le backend
   cd back
   npm run start:dev
2. Recuperer l'adresse ip du PC
   sur windows:
   ouvre l'invite commande et tapez ## ipconfig puis copiez l'adresse IPv4 sur le fichier url.js dans le dossier fron
